Result:
=======

Playable. Need ``libsdl2-dev`` installed.

.. figure:: pyrogue_screen.png
   :alt: pyRogue screenshot (level 1)
   :height: 641px
   :width: 976px
   :align: center

Controls:

* 'arrows' to move and attack
* 'g': grab thing
* 'i': show inventory
* '<': go downstairs (one way, there is no turning back!)
* 'ESC' to quit (or 'c' at main menu)
